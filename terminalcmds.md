# Terminal Commands

## **Terminal**

### What's in here?
```
ls
```
### Where am I?
```
pwd
```
### Show all the commands you have run
```
history
```
### Clear output 
```
cmd k
```

---
## **Directories**
### Change Directory
```
cd
```
### Create a new directory
```
mkdir
```
---
## **Files**
### Create a file 
```
touch 
```
### Remove a file
```
rm
```
### Rename a file
```
mv
```
